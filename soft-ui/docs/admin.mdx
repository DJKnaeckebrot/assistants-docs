---
title: "Admin"
description: "An Object which contains all the information for the admin page."
---

```js
theme: SoftUI({
  admin: {
    pterodactyl: {
      enabled: Boolean,
      apiKey: String,
      panelLink: String,
      serverUUIDs: String[],
    },
  },
});
```

## Example Usage

```js
theme: SoftUI({
  admin: {
    pterodactyl: {
      enabled: false,
      apiKey: "myPterodactylApiKey",
      panelLink: "https://panel.example.com",
      serverUUIDs: ["server-uuid-1", "server-uuid-2"],
    },
  },
});
```

## Properties

### pterodactyl

An Object which contains all the information for the pterodactyl section of the admin page.

### pterodactyl.enabled

A Boolean which disables the pterodactyl section.

### pterodactyl.apiKey

A String which is the API key for the pterodactyl panel. This MUST be a client api-key, found at /account/api

### pterodactyl.panelLink

A String which is the link for the pterodactyl panel.

### pterodactyl.serverUUIDs

An Array which is the UUID for every server you wish to control.
