---
title: "Footer"
description: "An Object which contains all the information for the footer."
---

```js
theme: SoftUI({
  footer: {
    replaceDefault: Boolean,
    text: String,
  },
});
```

## Example Usage

```js
theme: SoftUI({
  footer: {
    replaceDefault: true,
    text: "Bot developed by Plain",
  },
});
```

## Properties

### replaceDefault

A Boolean which replaces the default footer.

### text

A String which sets the text for the footer.
