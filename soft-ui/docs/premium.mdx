---
title: "Premium"
description: "An Object which contains all the premium information."
---

```js
theme: SoftUI({
  premium: {
    enabled: Boolean,
    card: {
      title: String,
      description: String,
      bgImage: String,
      button: {
        text: String,
        url: String,
      },
    },
  },
});
```

## Example Usage

```js
theme: SoftUI({
  premium: {
    enabled: true,
    card: {
      title: "Want more from Assistants?",
      description: "Check out premium features below!",
      bgImage:
        "https://assistantscenter.com/wp-content/uploads/2021/11/cropped-cropped-logov6.png",
      button: {
        text: "Become Premium",
        url: "https://patreon.com/sharkybot",
      },
    },
  },
});
```

## Properties

### enabled

A Boolean which changes whether or not the premium card is enabled.

### card

An Object which contains all the premium card information.

### card.title

A String which changes the card title.

### card.description

A String which changes the card description.

### card.bgImage

A String which is an image URL that changes the card background URL.

### card.button

An Object which contains the information for the button.

### card.button.text

A String which will change the button text.

### card.button.url

A String which will change the link of the button.
